初めてPCを分解し、メモリを増設することに成功した。3日経って毎日使っているが、現状不具合は起きていない。

同僚からアドバイスをもらったり、写真を撮ってClaudeに送って指示を受けたりはしたが、初めてにしてはよくやったと思う。勉強になったのでメモ。

# 使用PC

今回分解したPCはDellのLatitude 3440。

# 1. ベースカバーを外す

まずPCを裏返してベースカバーを見る。するとネジが9本あるので、プラスドライバーを使ってすべて緩める。

すべて緩めても、まったく空く気配は無い。

だが緩めたことで、ベースカバーとシャーシ（PCの本体フレーム・枠組み）の間に少しだけ隙間ができるので、そこにヘラ（スクライブ）を差し込み、少しずつ慎重に開いていく。

私はマイナスドライバーを使った↓

![](https://static.zenn.studio/user-upload/22598564d07a-20260627.webp)

そして隙間が大きい部分からヘラを差し込んで開けていった。だがどうやら公式いわく、ヒンジ（液晶画面と本体をつなぐ回転部分）周辺から開けていくのが正らしい。

> プラスチック製スクライブを使用して、ヒンジ近くにあるベース カバー下端のU字型のくぼみから順に、ベース カバーをこじ開けます。

![](https://static.zenn.studio/user-upload/057c13964cac-20260627.webp)

https://www.dell.com/support/manuals/ja-jp/latitude-14-3440-laptop/latitude_3440_sm/%E3%83%99%E3%83%BC%E3%82%B9-%E3%82%AB%E3%83%90%E3%83%BC%E3%81%AE%E5%8F%96%E3%82%8A%E5%A4%96%E3%81%97?guid=guid-7fe0ee34-fe90-4d3e-b93d-27910fae1110&lang=ja-jp


ベースカバーを剥がす際に、ベースカバーがシャーシと接合している部分である「爪」が外れることで、毎回「パキッ」という不可逆な重みが乗ったとても恐ろしい音が鳴るが、問題ない。丁寧に開けるというより無理やりこじ開ける感覚で、どんどんバキバキ鳴らして開けていこう。（後述のようになりたくなかったらゆっくり丁寧にやった方が良いかも）

# 2. PCの中身

ヘラでベースカバーとシャーシの隙間を全箇所抜け漏れなくこじ開けて初めて、パカっとベースカバーが取れる。ここでPCの中身、各種コンポーネントを見ることができる。

https://www.dell.com/support/manuals/ja-jp/latitude-14-3440-laptop/latitude_3440_sm/latitude-3440%E3%81%AE%E4%B8%BB%E8%A6%81%E3%81%AA%E3%82%B3%E3%83%B3%E3%83%9D%E3%83%BC%E3%83%8D%E3%83%B3%E3%83%88?guid=guid-d8923b1b-17a3-4f3c-afa1-8bb58d464e96&lang=ja-jp

見た目はこちらなどを参考に。

https://atluckneo.jugem.jp/?eid=1517

:::message
メモリスロットとは反対側のヒンジの近く、冷却ファンやボタン電池の上らへんにある灰色の金属部分に、埃が結構溜まっていたので除去した。分解したことにより外の埃が偶然舞って付いてしまったのか、この機械が埃を吸いやすい材質なのか、元々外から少しずつ入っていて溜まったのかはわからないが、気づけて良かった。
:::

下半分はどデカい横長メインバッテリーと左右端にスピーカー2基（L字型と⅃字型）があって、上半分は青いマザーボードに左から冷却ファン、SSD、CPU、ヒートパイプ、メモリスロットなどが接続されている。

下半分はバッテリーが大半を占めるが、上半分はごちゃごちゃしている（いわゆる"パソコンの中身"って感じ）という印象。

メインスロットは右上部分で、すでに緑色の16GBメモリが挿してあり、右隣りが空いていた。"DIMM2 DDR4 Only"と書いてあったので、意味を調べた。

## 余談1: メモリスロットのシルク印刷の意味

空きメモリスロットに白文字で"DIMM2 DDR4 Only"とシルク印刷されているが、それの意味を理解する。

まずDIMMとは何か。

https://www.ibm.com/jp-ja/think/topics/dimm

https://www.pc-koubou.jp/faq/faq_detail.html?id=20217&category=316&page=1&srsltid=AfmBOoqkpafpewrvUeiqhLUalkCOpLE29R9En3zgU-JyE8Mij7HYhqzl

https://e-words.jp/w/DIMM.html

DIMMは Dual Inline Memory Module の略で、メモリ規格、というかメモリの形状カテゴリの1つらしい。メモリ端の金色の金属端子が、表と裏で別の別の端子として機能するもの。他にもいろいろ規格はあるが、現代のRAMの主流はDIMM。

更にそのDIMMという規格・形状に従ったメモリにも世代が複数あって、DDR4やDDR5などがそれ。DDR5の方が新しいので、早くて高価。

https://e-words.jp/w/DDR4_SDRAM.html

https://e-words.jp/w/DDR5_SDRAM.html

で、今回は"DIMM2 DDR4 Only"と書いてあったので、「DIMM規格のスロットのうち2つ目のスロットで、DDR4世代のメモリだけ（Only）挿し込めますよ」という意味になる。この2つのメモリを挿す領域はメモリスロットとかDIMMスロットとか呼ばれる。

ちなみにDIMMの前世代にSIMMというのがある。後継版としてDIMMが生まれ、RAMデータの転送が高速になった。

https://e-words.jp/w/SIMM.html

## 余談2: SSDが意外に小さい

16GBしかないRAMより、1TB入るSSDの方が小さいのは意外だった。データ転送速度を犠牲にすれば、大容量でも物理サイズは小さく済むのだろう。ちなみに外付けハードディスクだとSSDではなくHDDなのでもっとデカい。

SSDは1つしか領域用意されてないので拡張はできないが、取り換えはできそう。だがSSDにはOS情報とか入ってるのでミスったらやばそう。バックアップなどは必須であろう。

（1 x の記述があるため1つしか領域がない）

> 1 x M.2 2230/2280ソリッドステート ドライブ

https://www.dell.com/support/manuals/ja-jp/latitude-14-3440-laptop/latitude_3440_ss/%E3%82%B9%E3%83%88%E3%83%AC%E3%83%BC%E3%82%B8?guid=guid-a00459e0-2a64-449c-9f02-581bf2e3c2cf&lang=ja-jp

# 3. メモリを挿す

:::message alert
ちなみにバッテリーに以下が記載されている。
> Disconnect the battery before accessing any parts or devices.
> Note: Longer POST time may occur if battery cable was disconnected for service/repair.

和訳は以下。
> 部品や機器に手を加える前に、バッテリーを取り外してください。
> 注：メンテナンスや修理のためにバッテリーケーブルを取り外した場合、POST時間が長くなることがあります。

つまりメモリとか内部装置いじるんならバッテリーを取り外せということらしい。だが面倒なのと余計な問題が発生したら対処し切れる自信が無いので省略した。本当はやった方が良さそう。

---

ちなみにこのPOSTというのは当然HTTP POSTではなくて、Power-On Self-Test の略で「電源投入時自己診断テスト」と呼ばれる。PC電源投入時やリセット時に実行される、OSの起動前にBIOSなどが行う様々な処理のようだ。

https://e-words.jp/w/POST.html

https://www.dell.com/support/contents/ja-jp/article/product-support/self-support-knowledgebase/fix-common-issues/no-post

> POSTとは、Power-On Self-Test（電源投入時の自己診断）の略です。これは、コンピューターの電源を入れるたびに、すべてが正常に機能していることを確認するために実行される簡単なチェックのようなものです。

なのでまぁバッテリーを挿し直したらPOST時間が長くなり得るというのは納得できる。

:::

---

とりあえず、"DIMM2 DDR4 Only"とシルク印刷で書かれてある空きスロットにメモリを挿してメモリ増設をしていく。

（後から気づいたが、Dell公式がメモリの取り外し方を説明しているサイトがあった）

縦長の8GBメモリを入れるにあたって、縦の一方の側面が金色の金属端子になっており、そこを左隣のすでに挿さっているメモリ側に挿し込む。

表裏の判別方法については、メモリの金属端子側の中央部くらいに小さな穴が開いており、そこに丁度フィットするようにメモリスロット側も中央部に突起があった。（穴と突起は逆だったかも）

なのでそれがハマる方が正しい表裏になる。

入れ方だが、先ほどのDell公式ページがわかりやすい。

![](https://static.zenn.studio/user-upload/9ef9aa5572fd-20260627.jpg)

縦長のものを左横に挿し込む訳だが、左横部分が下に傾く（右横が上に傾く）ようにして中に入れ、カチっというまで奥に挿し込むのが大事。

自分は同僚から、しっかり挿さなかった結果1週間ほど突然シャットダウンされたりして不安定になり挿し直すハメになったという話を聞いていたのと、Claudeが「音が鳴るまで強く挿し込むのが大事」と言っていたので、「カチッ」となるまで強く挿し込んだ。この話を聞いていなければ「こんなに強く押し込んで大丈夫か？」と不安になっていたかもしれない感覚だった。感謝。

音が鳴ったあとは、傾けた角度を既に挿してある左隣のメモリと同じ角度になるように戻す。

これでメモリ増設が完了。

# 4. ベースカバーをはめ直す

外したベースカバーをシャーシに宛て、プラスドライバーでネジをすべて締める。

そして全箇所の爪をカチっと言うまで手で押し込む。

これで分解作業は終了。

# 5. メモリ確認

ちゃんとメモリ増設ができているか、PCに新しいメモリが認識されているか、最後に確認する。

だが電源を入れるとすぐBIOSが起動してメモリ増設を認識してくれていることが確認できる。「Continue」でOK。

![](https://static.zenn.studio/user-upload/9d69094a1efa-20260627.png)

OS起動後、タスクマネージャー > パフォーマンス > メモリ からメモリが16GBから24GBに変わっているため、成功！

![](https://static.zenn.studio/user-upload/b01093608265-20260627.png)

5秒くらいかかるが、bashやpowershellで`systeminfo`を実行してメモリの部分を確認しても良い。

# 感想など

今までソフトウェアの経験しかなかったので、ハードウェアにはずっと憧れがあった。だがいろいろと敷居が高いと勝手に思っていた。

- 物理的な購入が必須で金がかかる
- 心理的ハードル
    - ハードを買ったとてぶっ壊してしまったら詰むのではないか
    - 自分と同じPCで同じことをやろうとしていて説明が丁寧なサイトは果たしてあるだろうか
    - ...という不安はソフトウェアだったらある程度解消できる。ハードウェアと違ってソフトウェア領域ではコピペが容易だし、試行錯誤が容易にたくさんできるし、写真を撮る必要もないし、情報がありふれている
- 普段の生活から遠い領域
    - ソフトウェアは普通に生きていてもブラウザでWebサイトに触れたりアプリケーションを入れたりするので自然に身近な存在だが、ハードウェア（特にPCの内部部品など）にしっかり対面する機会は普通に生きていたら少ない

だが同僚がメモリを増設したという話を聞いてやってみようと思った。

あとは

- Claude Codeで重いタスクを並列で動かす
- ブラウザで多めにタブを開く
- VSCodeで重いフォルダを開く

などを同時にやっていると画面が固まることがあった。

固まらずともタスクマネージャーを見るとメモリが90%前後まで使用されていることはざらだったので心理的不安もあった。

結果的に、普段から触れているPC、使っている/作っているソフトウェア、そういうものの土台がどういう形をしているのか見ることができて、とても良い経験になった。

（正直ハードウェアをまったく知らないのにソフトウェアをやっていると言うのが恥ずかしい気持ちがあったのだが、これでやっと少しは自信を持てた気がする）

今回挿したメモリは8GBだが、各メモリは32GBまで対応しているようなので、今後メモリ不足になったら32GBメモリを購入し、それと今回挿した8GBメモリを交換してみようと思う。既に挿さっていたDIMM1の方の16GBメモリを32GBに取り換えるのも良いかもしれない。
